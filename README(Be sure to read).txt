Use this program to search in the Shodan engine
 
                                                       "Install prerequisites"
                                                       
Install the coloroma library By order:
                                              pip install coloroma
                                              
Install the shodan library By order:
                                              pip install shodan
                                              
                                                       "Changes in the program"
                                                       
You can change the SHODAN_API_KEY variable , Enter your API key on the Shodan site and search using your account
